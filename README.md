# DOREA-X  
**Document-Oriented Reasoning and Explanation Assistant – eXtended**

> 연구와 업무 현장에서 즉시 활용 가능한 차세대 문서 이해 AI 플랫폼

---

## 🔎 Overview

**DOREA-X**는 기존 DOREA를 기반으로 완전히 재설계된 차세대 지능형 문서 이해 시스템입니다.  
단순한 PDF 챗봇을 넘어, 연구 및 업무 환경에서 즉시 활용 가능한 **실용 중심 멀티모달 RAG 플랫폼**을 목표로 합니다.

---

## 🚀 Key Features

- **Production-Ready Architecture**  
  실서비스 운용을 전제로 한 재설계 구조

- **Multi-User Support**  
  다중 사용자 세션 관리 및 협업 환경 지원

- **Multimodal RAG**  
  텍스트·표·이미지를 통합 이해하는 문서 기반 추론

- **Extensible OCR & Layout Integration**  
  다양한 OCR 및 문서 레이아웃 인식 서비스와 연계 가능한 확장 구조

- **Flexible LLM Support**  
  OpenAI GPT 및 로컬 LLM(Ollama 등) 유연 지원

- **Real-Time Streaming Response**  
  스트리밍 기반 대화형 응답

- **Conversation & Project Management**  
  대화 기록 저장 및 프로젝트 단위 문서 관리

---

## ✨ What Makes DOREA-X Different

- 기존 시스템의 단순 개선이 아닌 **제로부터 재설계**
- 특정 OCR 또는 모델에 종속되지 않는 **서비스 연계형 구조**
- 연구 프로토타입이 아닌 **현장 적용 중심 플랫폼**
